# macDisplayBar

macDisplayBar is an open-source macOS menu bar app for controlling displays without opening System Settings.

It is built for people who switch resolutions often, manage multiple monitors, or want quick access to brightness, dimming, HDR, color, arrangement, and advanced display controls from one compact menu.

## Features

- Menu bar only app with a compact popover UI
- Display picker for switching between connected monitors
- Brightness and dimming controls
- Resolution picker with search, filters, HiDPI labels, and favorites
- Automatic HiDPI candidates for exposed resolutions at the highest refresh rate
- 30-second revert prompt after resolution changes
- HDR, color, and extra brightness controls where supported
- Display arrangement controls for multi-monitor setups
- Advanced display information and compatibility tools
- In-app update check powered by Hot Updater
- Korean and English UI text

Some controls depend on macOS, the connected display, cable type, and monitor firmware. Unsupported controls are hidden or disabled where possible.

When macOS exposes a resolution without a matching HiDPI option, macDisplayBar shows an additional HiDPI candidate for that resolution using the highest available refresh rate. Selecting one creates the matching display override settings file; macOS may require installing that override and restarting the display session before the new mode becomes selectable.

## Install

### Homebrew

```sh
brew install --cask jingjing2222/tap/mac-display-bar
```

Then launch `macDisplayBar` from Applications. The app runs in the menu bar.

### From Source

Requirements:

- macOS 14 or newer
- Xcode
- CocoaPods
- Node.js 20 or newer
- Yarn 4

Clone and install dependencies:

```sh
git clone https://github.com/jingjing2222/mac-display-bar.git
cd mac-display-bar
yarn install
pod install --project-directory=macos
```

Run in development:

```sh
yarn start
```

In another terminal:

```sh
yarn macos:debug
```

Build a release app locally:

```sh
yarn macos:release
```

The built app is generated by Xcode under DerivedData.

## Updates

Use the update button at the top of the popover. macDisplayBar checks for an available app bundle update and applies it with a reload when one is ready.

## Development

Common checks:

```sh
yarn test
yarn tsc --noEmit
yarn lint
yarn format:check
yarn check:fonts
```

Hot update bundles use fingerprint matching:

```sh
yarn hot-updater fingerprint create
```

## Project Scope

macDisplayBar is macOS-only. iOS and Android app targets are intentionally not part of this project.
